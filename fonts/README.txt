==== 自定義字體目錄 ====

此目錄用於放置自定義的中文字體文件，以便程式能夠正確顯示繁體中文文字。

建議放置的字體：
1. 微軟正黑體 (Microsoft JhengHei) - 適合繁體中文顯示
2. 標楷體 (DFKai-SB) - 傳統中文字體
3. 細明體 (PMingLiU) - 常用於印刷品
4. 黑體 (SimHei) - 簡體中文字體，但也可顯示繁體中文

如果您在圖表或報告中看到關於缺少字體的警告，請將上述字體文件放置在此目錄中，程式將自動載入這些字體。

字體文件通常位於：
- Windows系統：C:\Windows\Fonts
- Mac系統：/Library/Fonts 或 ~/Library/Fonts
- Linux系統：/usr/share/fonts 或 ~/.fonts

請確保您有權使用和分發這些字體文件。
